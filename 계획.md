# FE
### 로그인
DB에 본인 아이디 1개만 만들어 놓고, 로그인

로그인하면 모든 기능들 사용가능

### 기능

1. Todo
- 오늘의 남은 시간 타이머
- 할 일 추가
- 할 일 삭제
- 할 일 수정
- 유저의 데이터에서 할 일 목록 불러오기
- DONE, NOT DONE 으로 구분

2. 게시판
- 게시글 추가
- 게시글 수정
- 게시글 삭제
- 게시글 목록 불러오기
- 페이지네이션
- 게시글 카테고리 분류 (JS인지, React인지, 알고리즘인지)

# BE
1. 기본세팅 (express, mongoose, app.listen() 세팅)
2. 라우터 주소 정의
3. 데이터베이스 Schema 정의
4. 기능정의 (CRUD)
5. 테스트 (Postman 을 이용한 테스트)
